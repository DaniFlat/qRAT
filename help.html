Everything you need to know about qPCR Relative Expression Analysis with qRAT

What's qRAT?

qRAT is an R-based standalone application designed to automate the processing of raw Quantification Cycle (Cq) data files. It works with data exported from virtually any qPCR instrument using well-established and state-of-the-art statistical and graphical techniques.

The purpose of this tool is to provide a comprehensive, straightforward, and easy-to-use solution for the relative quantification of RT-qPCR data that requires no programming knowledge.

Amplification Efficiencies
For ∆∆Cq validity, efficiencies of target and reference primers must be approximately equal and close to 1 (100%), as stated by Livak and Schmittgen.
Quick Checklist
Export Cq values in csv or txt format
The data area must start with the word "Well"
Required columns: Well, Sample, Gene, Cq
Tech. replicates must have the exact same name
Inter-Plate Calibrator sample name must be the same on all plates
Avoid special characters or spaces in names
